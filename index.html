Como Jogar

Use as teclas A e D ou as setas ← e → para mover o Pou para os lados.

Pegue as comidinhas que caem para ganhar pontos:

Laranja: +10 pontos Maçã: +20 pontos Uva: +30 pontos

Evite deixar as comidinhas caírem no chão para não perder vidas!

O jogo termina quando o tempo acaba ou você perde todas as vidas.
